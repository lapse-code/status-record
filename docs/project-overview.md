# Project Overview

## 产品定位

Status Record 是一个面向个人学习和深度工作的网页工具。第一版目标不是做复杂的任务管理，而是用一个低摩擦的番茄钟流程，把每天真正发生的专注、拖延、注意力切换、产出、不专注原因和睡眠状态记录下来，并能按日、周、月复盘。

## 核心问题

用户希望避免主观美化记录，尤其是拖延和专注时长。因此系统应尽量使用计时器和时间戳生成客观数据，只在必须人工判断的地方使用标签和短文本。

## 第一版用户流程

1. 用户到工位或学习位置后，可以先点击“到岗/开始记录”，系统开始记录到岗时间，但不算专注时长。
2. 用户真正开始专注时，启动固定或自定义番茄钟。
3. 如果用户未提前点击到岗而直接启动番茄钟，系统会同步创建到岗记录并绑定本轮专注，拖延记为 0。
4. 番茄钟结束后，系统弹出复盘表单：
   - 选择本轮状态。
   - 输入注意力切换次数。
   - 记录可见产物标签和文字。
   - 选择不专注原因标签。
   - 选择是否使用累计休息时间。
5. 系统把本轮专注时长、拖延、注意力切换、产物、不专注原因和状态写入记录。
6. 如果用户忘记开启番茄钟，可以用“手动记录”补录开始时间、持续分钟和复盘参数；它进入专注/不专注统计和今日休息累计，但不创建、关闭或重置到岗记录。
7. 用户每天记录一次睡眠时长和主观精力，之后可修改。
8. 用户查看日、周、月统计。

## 每天记录的指标

| 指标 | MVP 记录方式 | 统计方式 |
| --- | --- | --- |
| 专注时长 | 根据完成的番茄钟、自定义计时器或手动记录累计 | 日、周、月总分钟数，按状态/标签拆分 |
| 拖延 | 从到岗区间中扣除专注、休息和不专注后的等待时间自动计算 | 日、周、月总分钟数和平均值；与点阵红色真实时长同源 |
| 注意力切换次数 | 每轮结束后输入一个数字 | 日、周、月总次数和每小时次数 |
| 可见产物 | 标签 + 短文本 | 标签频次、产物列表、按时间回顾 |
| 不专注原因 | 标签选择，支持“无”和自定义标签 | 标签频次、趋势、与专注时长的关系 |
| 睡眠 | 每天一次：15 分钟步进的睡眠时长 + 主观精力 1-5 | 日记录、周/月均值、与学习表现对照 |

## 倒计时结束后的五种状态

五种状态应作为可扩展标签保存，MVP 可以提供以下默认值：

| 状态 | 含义 |
| --- | --- |
| 完成 | 本轮基本按计划完成 |
| 部分完成 | 做了，但没有达到预期范围 |
| 被打断 | 外部事件导致中断或效果下降 |
| 卡住 | 主要时间花在理解、调试、搜索或犹豫上 |
| 放弃/转移 | 本轮没有继续原计划，改做别的或停止 |

后续如果用户觉得这五类不准确，应允许在设置里改名、改颜色、归档或新增。默认“完成”状态的固定 id `status-completed` 是专注判定条件，可以改名和改颜色，但不能归档或删除；其他已归档标签不再出现在新复盘选择中，但历史记录继续保留。

## 休息时间规则

- 系统维护一个“每日学习累计账本”：当天所有已完成番茄钟的真实有效分钟会累计到同一个账本。
- 今日学习累计每满 25 分钟，系统自动给今日休息余额增加 5 分钟；不要求单轮番茄钟完整满 25 分钟。
- 休息余额和未满 25 分钟的累计进度只在当天有效，第二天重新从 0 开始。
- 自定义倒计时、提前完成和完整完成都会按真实有效番茄钟分钟写入当天累计账本；取消本轮不写入累计账本，不产生休息余额。
- 每轮结束后询问用户是否现在使用休息时间。
- 用户可以选择：
  - 使用一部分或全部休息余额，并启动休息倒计时。
  - 保留休息余额，到岗记录保持打开，等待用户选择下一轮番茄钟。
- 休息倒计时可以提前结束；提前结束时，未用完的分钟退回休息余额。
- 休息自然结束后，如果仍有可用休息余额，系统弹窗询问是否继续休息；可选继续 5 分钟、使用全部剩余休息、自定义分钟，最大不能超过当前余额。
- 用户选择开始专注，或没有剩余休息余额时，系统回到番茄钟选择；如果到岗仍打开，休息之外的等待继续显示为拖延。只有用户点击“离开”才关闭到岗周期。

## 时间线和时区规则

- 系统同时保存 UTC 时间戳、记录发生时区下的 `local_date` 和 IANA `time_zone`。
- 新记录使用当前设备时区；旧数据或旧备份缺少时区时按 `Asia/Tokyo` 兼容。
- 日点阵和周点阵按记录发生时区绘制，用户旅行或切换电脑时区后，历史记录不会被重新排布成另一个国家的时间线。
- 未来如果需要，可以再增加“按当前位置时区查看历史”或“固定某个时区查看历史”的显示模式；MVP 默认使用记录发生时区。

## MVP 范围

- 网页应用。
- 本地优先存储。
- 固定倒计时和自定义倒计时。
- 到岗记录和拖延自动计算。
- 手动记录，用于补录忘记开启番茄钟但已经完成的学习时间。
- 每轮结束复盘。
- 休息倒计时和休息结束提醒。
- 设置页：点阵颜色、标签新增、改名、改色、归档、解除归档和删除。
- 睡眠每日记录。
- 日、周、月统计。
- 一天 5 分钟粒度的点阵视图。
- 一周 5 分钟粒度点阵视图，用于按周查看每天时间分布。
- 数据导出/导入，至少支持 JSON，后续可加 CSV。
- 数据操作入口在大屏侧栏显示；当侧栏操作区被隐藏时，必须在主内容区域显示同等入口。

## 暂不做

- 多人协作。
- 云同步。
- 复杂任务管理。
- 自动检测手机或应用切换。
- 精确自动测量深度工作分钟。
- Mac/iOS 原生壳，等网页版本稳定后再评估。
